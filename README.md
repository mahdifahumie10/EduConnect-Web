# EduConnect-Web

# DESCRIPTION 
- Developed a simple web application with the use of Visual Studio 2022's .NET Framework (Web Forms) to create an educational website that provides students and lecturers role specific features such as viewing/uploading notes, etc. 

# FEATURES
- Role Based Login - Users are directed to specific pages based on their role (Lecturer, Student)
- Common home page for guest users/no login
- Frontend/Backend implementation - Database using SQL to store data like student/lecturer login credentials,HTML/CSS to create the web page frontend layout, C# to connect frontend to backend logic.
- Features such as adding/deleting/viewing notes, quizzes, CRUD operation implementation
  
# HOW TO RUN
- Click the green Code button and Download ZIP
- Recomended - Extract the necessary folders, navigate to EduConnect.sln and open using Visual Studio 2022 (there are quite a few zip files due to file size)
- Note - Some functionality (particularly deleting notes under lecturer role) may not work, the website is still under construction :)
